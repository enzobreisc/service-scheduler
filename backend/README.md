# Service Scheduler - Backend (API)

API desenvolvida em Laravel para gerenciar agendamentos, serviços e usuários.

## Configuração

1. Instale as dependências: `composer install && npm install`
2. Configure o `.env` (já deve estar pronto se você seguiu o setup principal).
3. Gere a chave: `php artisan key:generate`
4. Execute migrations: `php artisan migrate`
5. Inicie o servidor: `php artisan serve`

## Tecnologias

- Laravel 11
- SQLite
- Sanctum
